🎯 運行順序（一步一步跟住做）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 1️⃣：打開終端機
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果你用緊 WSL (Linux)：
  • 打開 WSL 終端機
  • 或者 Windows Terminal

如果你用緊 Windows：
  • 打開 PowerShell
  • 或者 CMD


步驟 2️⃣：進入 ming 目錄
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WSL 環境：
  cd /mnt/c/Users/user/Desktop/ming/ming

Windows PowerShell：
  cd C:\Users\user\Desktop\ming\ming

檢查你喺邊度：
  pwd          (WSL)
  或者
  Get-Location  (PowerShell)


步驟 3️⃣：檢查 Python
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WSL：
  python3 --version

Windows：
  python --version
  或者
  py --version

如果顯示版本號，就代表 OK ✅


步驟 4️⃣：檢查故事檔案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WSL：
  ls -la my_custom_story.txt
  或者
  ls -la ../my_custom_story.txt

Windows：
  dir my_custom_story.txt
  或者
  dir ..\my_custom_story.txt

應該要見到檔案 ✅


步驟 5️⃣：運行腳本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WSL：
  python3 run_custom_story.py

Windows：
  python run_custom_story.py

║                      🔧 完整命令（一次過複製）                            ║
╚══════════════════════════════════════════════════════════════════════════╝

WSL 環境（複製呢三行）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

cd /mnt/c/Users/user/Desktop/ming/ming
python3 run_custom_story.py


Windows PowerShell（複製呢三行）：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

cd C:\Users\user\Desktop\ming\ming
python run_custom_story.py

錯誤 1：找不到 Python
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WSL：
  sudo apt update
  sudo apt install python3 python3-pip

Windows：
  去 python.org 下載安裝 Python


錯誤 2：找不到模組
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

cd /mnt/c/Users/user/Desktop/ming
pip3 install -r requirements.txt


錯誤 3：找不到故事檔案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

確保檔案喺：
  C:\Users\user\Desktop\ming\my_custom_story.txt
  或者
  C:\Users\user\Desktop\ming\ming\my_custom_story.txt
